# 安装Git和Arduino IDE (Windows) {docsify-ignore-all}

## 1. 安装`Git`
如果你已经安装了`Git`，请直接从第2步开始。

否则点击这里下载[Git](https://git-scm.com/download/win)并安装它

## 2. 安装`Arduino IDE`

*下载地址*
https://www.arduino.cc/en/Main/Software
